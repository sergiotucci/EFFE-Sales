# EFFE Ring PWA

Mini gestionale mobile-first per ordini e magazzino EFFE Ring.

## Uso su iPhone

1. Pubblica la cartella su GitHub Pages oppure aprila da un piccolo hosting web.
2. Apri `index.html` da Safari.
3. Tocca Condividi → Aggiungi a Home.

## Funzioni

- Ordini da mobile
- Magazzino per taglia e colore
- Controllo disponibilità
- Alert riordino
- Export CSV per Excel/Numbers
- Backup JSON
- Funziona offline dopo il primo caricamento
